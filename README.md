# Starchess
_by Screeper11_

## Setup
1. Install [bun](https://bun.sh/)
  
**Note:** Must be installed on a Linux machine
 
```sh
curl https://bun.sh/install | bash
```

2. Run `bun setup`

## Start
### VS Code
Start "ALL" compound

### Manual
Start all scripts in seperate terminals
1. `bun client`
2. `bun auth`
3. `bun game`
